# See-Crits

A Python implimentation of Critters for OOP and programming pratice

# Installation

git clone https://github.com/DanielTheSilly/See-Crits.git

on Windows:
python -m venv venv
venv\Scripts\activate

on Mac/Linux
python3 -m venv venv
venv\bin\activate

on both:
pip install -r requierments.txt

# Running

(from folder)
python SeeCrits.py
